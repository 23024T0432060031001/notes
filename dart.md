#### Command line

###### To check existing setup

```sh
flutter doctor
```

###### To upgrade flutter and dart

```sh
flutter upgrade
```

###### To run the app on an emulator or an actual device

```sh
flutter run
```

###### To build APK

```sh
flutter build apk
```

###### TO build iOS

```sh
flutter build ios
```
